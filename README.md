# 🏢 Perusahaan Indonesia

Daftar lengkap perusahaan besar di Indonesia berdasarkan kategori — Bank, Telkom, E-commerce, Teknologi, dan banyak lagi.

## 🚀 Live Demo

👉 [https://syaptiyan.github.io/perusahaan-indo](https://syaptiyan.github.io/perusahaan-indo)

## ✨ Fitur

- **50+ Perusahaan** — data lengkap perusahaan besar Indonesia
- **12 Kategori** — Bank, Telkom, E-commerce, Teknologi, F&B, Otomoif, dll
- **Filter per Kategori** — klik kategori untuk filter
- **Pencarian** — cari berdasarkan nama atau deskripsi
- **Detail Modal** — CEO, pendiri, tahun berdiri, kantor pusat, status
- **Dark Theme** — desain modern dan elegan
- **Responsive** — tampil sempurna di mobile dan desktop

## 🛠️ Teknologi

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Glassmorphism)
- JavaScript
- [Alpine.js](https://alpinejs.dev/) — reactive framework
- [Font Awesome](https://fontawesome.com/) — icons
